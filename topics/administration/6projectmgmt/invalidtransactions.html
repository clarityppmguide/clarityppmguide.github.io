Invalid Transactions

This page lists the Transactions there are failed due to missing value for a field required for Transactions [OR] incorrect value being set-up for that field.

These failed or rejected transactions can be executed again from the same page itself.

Navigation

Please follow the below steps

  • Navigate to Administration tab
  • Under Project Management click on Invalid Transactions link

Click the images to see the view

Access Rights

For Developers

Access Rights are considered as permission to view or access particular section in CA PPM

Administration - Application Setup Allows user to edit PPM options and settings, including OBS, Time, Data Administration and General Settings. Includes the Administration - Navigate right.

Tables

For Developers

Below are the tables used for this View - Click the Image to view.

1.IMP_TRANSACTIONIMPORT This table stores the Calendars created in the CA PPM application.

Query

For Developers

Following Query is used to fetch the data from View 


                    SELECT

                    TO_CHAR(timp.JOB_ID) AS job_id,
                    timp.TRANSDATE AS transaction_date,
                    i.code AS ProjectID,
                    r.full_name AS resource_name,
                    transstat.NAME AS import_status_desc,
                    err.DESCRIPTION AS error_description

                    FROM IMP_TRANSACTIONIMPORT timp
                    LEFT JOIN SRM_PROJECTS prj ON prj.UNIQUE_NAME = timp.PROJECT_CODE
                    INNER JOIN inv_investments i
                               on i.id = prj.id
                    LEFT OUTER JOIN cmn_lookups_v transtype ON timp.transtype = transtype.lookup_code AND transtype.lookup_type = 'RPT_TRANSACTION_TYPE' AND transtype.language_code = 'en'
                    LEFT OUTER JOIN cmn_lookups_v transstat ON timp.IMPORTSTATUS = transstat.lookup_code AND transstat.lookup_type = 'TD_IMP_TRAN_STATUS' AND transstat.language_code = 'en'
                    LEFT OUTER JOIN prtask t ON timp.TASKID = t.prid
                    LEFT    OUTER JOIN cmn_lookups_v tname ON tname.lookup_type = 'COP_PORTLET_QUERY_TRANSLATIONS' AND tname.language_code = 'en'
                                       AND tname.lookup_code = 'Effort'
                    LEFT OUTER JOIN IMP_TRANSACTIONIMPORTERRORS err ON err.ERRORCODE = timp.ERRORCODE
                    LEFT OUTER JOIN srm_resources r ON timp.resource_code = r.unique_name
                    LEFT OUTER JOIN transclass tc ON timp.transclass = tc.transclass
                    LEFT OUTER JOIN prchargecode cc ON timp.charge_code = cc.prexternalid
                    WHERE   ((timp.transtype IS NULL) OR ( exists (select  timp.transtype  from dual) ))
                    AND    ((timp.IMPORTSTATUS IS NULL)   OR ( exists (select timp.IMPORTSTATUS from dual) ))
                    AND    ((r.unique_name IS NULL)   OR ( exists (select r.unique_name from dual) ))
