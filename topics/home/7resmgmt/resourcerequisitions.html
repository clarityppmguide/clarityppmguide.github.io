Resource Requisitions

A requisition is a request to staff a specific project with labor resources. The chief benefit of requisitions is planning

With requisitions, a project manager requests resources and a resource manager assigns them to investments. As a project manager, you can create simple or detailed requisitions that request resources for multiple time periods.

Navigation

Please follow the below steps

  • Navigate to Home tab
  • Under Resource management click on Resource Requisitions link

Click the images to see the view

Access Rights

For Developers

Access Rights are considered as permission to view or access particular section in CA PPM

Resource Navigate Allows user access to the Resource list and Resource Finder pages. The user will only be able to view resource information for resources to which the user has access."

Tables

For Developers

Below are the tables used for this View - Click the Image to view.

1.RSM_REQ_REQUISITIONS This table stores the resource requisitions created for staffing resources to projects.

Query

For Developers

Following Query is used to fetch the data from View 


SELECT
   		 REQ.NAME  Requisition
   		,REQ.IS_UNBOOK  Unbook
   		,REQ.PRIORITY_CODE Priority
   		,REQ.DUE_DATE DueDate
   		,(SELECT FULL_NAME FROM SRM_RESOURCES SRM WHERE SRM.USER_ID = REQ.BOOK_MANAGER_ID ) Manager
   		,(SELECT FULL_NAME FROM SRM_RESOURCES SRM WHERE SRM.USER_ID = REQ.REQUESTED_BY ) RequestedBy
   		,(SELECT NAME FROM INV_INVESTMENTS INV WHERE INV.ID=REQ.OBJECT_ID ) Investment
      ,REQ.REQUEST_START  Start
      ,REQ.REQUEST_FINISH  Finish
FROM

      RSM_REQ_REQUISITIONS REQ
