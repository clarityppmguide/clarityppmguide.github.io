Posted Transaction Review

The Posted Transaction Review portlet displays financial transactions across investments by Transaction date and includes quantity, cost, and currency information.

Use this portlet to compare transaction currency amounts to converted currency amounts by task and by resource

Navigation

Please follow the below steps

  • Navigate to Home tab
  • Under Financial management click on Posted Transaction review link

Click the images to see the view

Access Rights

For Developers

Access Rights are considered as permission to view or access particular section in CA PPM

Instance Level rights

"Give the access to Page - Under Instance rights--> Click Add--> select Page from Dropdown --> Posted transaction review

Give the access to Portlet - Under Instance rights--> Click Add--> select Portlet from dropdown --> Add by the Portlet name"

Tables

For Developers

Below are the tables used for this View - Click the Image to view.

1.PPA_WIP This table stores all primary transactions (Labor, Materials, Equipment, and Expense).
2.INV_INVESTMENTS This table stores CA Clarity PPM-defined data common to all investments ( Projects, Products, Assets, Applicatins etc,.).
3.PRTASK This table is used to generate lists of tasks and to record and present information on tasks in various views.
4.SRM_RESOURCES This table stores the resource related information. A row is created in this table when a new resource is added in the Application.
5.PPA_WIP_VALUES This table stores multi-currency values for rows in the PPA_WIP table.
6.PRCHARGECODE This table stores data on charge codes.
7.TRANSCLASS This table stores the transaction class information.
8.PRTYPECODE This table stores the names of type codes.
9.ENTITY This table stores financial entity details.
10.LOCATIONS This table stores the financial locations.
11.DEPARTMENTS This table stores the department details for financial properties.
12.WIPCLASS This table stores the work in progress (WIP) class information.
13.PAC_FOS_RESOURCE_CLASS This table stores resource class information.
14.PROJCLASS This table stores information about the project class.
15.CMN_LOOKUPS_V This table stores the the values for a static lookup type.
16.PPA_WIPAPINFO This table stores supplemental accounts payable (AP) information for transactions imported from an AP system.

Query

For Developers

Following Query is used to fetch the data from View 


SELECT
                      wip.transno wip_transno,
                    	wip.transdate wip_transdate,
                    	inv.id investment_int_id,
                    	inv.code investment_id,
                    	inv.odf_object_code inv_object_code,
                    	UPPER(inv.odf_object_code) inv_object_code_upper,
                    	inv.name investment_name,
                    	t.prid task_int_id,
                    	t.prexternalid task_id,
                    	CASE WHEN t.prexternalid = '~rmw' THEN tname.name ELSE t.prname END task_name,
                    	CASE WHEN t.prexternalid = '~rmw' THEN 0 ELSE 1 END task_disable,
                    	t.prwbssequence task_wbs_sequence,
                    	cc.prid chargecode_int_id,
                    	cc.prname chargecode_name,
                    	r.id resource_int_id,
                    	r.unique_name resource_id,
                    	r.full_name resource_name,
                    	tc.id transclass_int_id,
                    	tc.description transclass_name,
                    	tyc.prid inputtype_int_id,
                    	wip.input_type inputtype_name,
                    	wip.user_lov1 user_value1_id,
                    	userlov1.name user_value1_name,
                    	wip.user_lov2 user_value2_id,
                    	userlov2.name user_value2_name,
                    	wip.notes notes,
                    	wip.quantity quantity,
                    	prj_ent.ENTITY prj_entity_id,
                    	prj_ent.shortdesc prj_entity_name,
                    	res_loc.locationid res_location_id,
                    	res_loc.shortdesc res_location_name,
                    	res_dept.departcode res_dept_id,
                    	res_dept.shortdesc res_dept_name,
                    	prj_loc.locationid prj_location_id,
                    	prj_loc.shortdesc prj_location_name,
                    	prj_dept.departcode prj_dept_id,
                    	prj_dept.shortdesc prj_dept_name,
                    	wipclass.wipclass wipclass_id,
                    	wipclass.shortdesc wipclass_name,
                    	resclass.resource_class resource_class_id,
                    	resclass.description resource_class_name,
                    	prjclass.PROJCLASS prjclass_id,
                    	prjclass.shortdesc prjclass_name,
                    	wip.transtype transtype_id,
                    	transtype.NAME transtype_name,
                    	wip.cost_type costtype_id,
                    	costtype.name costtype_name,
                    	wip.chargeable chargeable,
                    	SUM(CASE WHEN wv.currency_type = 'HOME' THEN wv.totalcost END) home_totalcost,
                    	SUM(CASE WHEN wv.currency_type = 'HOME' THEN wv.totalamount END) home_amt,
                    	MAX(CASE WHEN wv.currency_type = 'HOME'THEN wv.currency_code END) home_curr,
                    	SUM(CASE WHEN wv.currency_type = 'HOME' THEN wv.rate_exchange_rate END) home_rate,
                    	SUM(CASE WHEN wv.currency_type = 'NATURAL' THEN wv.totalcost END) natural_totalcost,
                    	SUM(CASE WHEN wv.currency_type = 'NATURAL' THEN wv.totalamount END) natural_amt,
                    	MAX(CASE WHEN wv.currency_type = 'NATURAL'THEN wv.currency_code END) natural_curr,
                    	SUM(CASE WHEN wv.currency_type = 'NATURAL' THEN wv.rate_exchange_rate END) natural_rate,
                    	SUM(CASE WHEN wv.currency_type = 'REPORTING' THEN wv.totalcost END) rpt_totalcost,
                    	SUM(CASE WHEN wv.currency_type = 'REPORTING' THEN wv.totalamount END) rpt_amt,
                    	MAX(CASE WHEN wv.currency_type = 'REPORTING'THEN wv.currency_code END) rpt_curr,
                    	SUM(CASE WHEN wv.currency_type = 'REPORTING' THEN wv.rate_exchange_rate END) rpt_rate,
                    	SUM(CASE WHEN wv.currency_type = 'BILLING' THEN wv.totalcost END) bill_totalcost,
                    	SUM(CASE WHEN wv.currency_type = 'BILLING' THEN wv.totalamount END) bill_amt,
                    	MAX(CASE WHEN wv.currency_type = 'BILLING'THEN wv.currency_code END) bill_curr,
                    	SUM(CASE WHEN wv.currency_type = 'BILLING' THEN wv.rate_exchange_rate END) bill_rate
FROM                        PPA_WIP wip
                    LEFT    OUTER JOIN inv_investments inv ON inv.id = wip.investment_id
                    LEFT    OUTER JOIN prtask t ON wip.task_id = t.prid
                    LEFT    OUTER JOIN srm_resources r ON wip.resource_code = r.unique_name
                    LEFT    OUTER JOIN ppa_wip_values wv ON wip.transno = wv.transno and wv.currency_type <> 'EURO'
                    LEFT    OUTER JOIN prchargecode cc ON wip.charge_code = cc.prexternalid
                    LEFT    OUTER JOIN transclass tc ON wip.transclass = tc.transclass
                    LEFT    OUTER JOIN prtypecode tyc ON wip.input_type = tyc.prexternalid
                    LEFT    OUTER JOIN entity prj_ent ON wip.entity = prj_ent.entity
                    LEFT    OUTER JOIN locations res_loc ON wip.emplyhomelocation = res_loc.locationid
                    LEFT    OUTER JOIN departments res_dept ON wip.emplyhomedepart  = res_dept.departcode
                    LEFT    OUTER JOIN locations prj_loc ON wip.project_location = prj_loc.locationid
                    LEFT    OUTER JOIN departments prj_dept ON wip.project_department = prj_dept.departcode
                    LEFT    OUTER JOIN wipclass wipclass ON wip.wipclass = wipclass.wipclass
                    LEFT    OUTER JOIN pac_fos_resource_class resclass ON WIP.resource_class = resclass.resource_class
                    LEFT    OUTER JOIN projclass prjclass ON wip.projectclass = prjclass.projclass
                    LEFT    OUTER JOIN cmn_lookups_v tname ON tname.lookup_type = 'COP_PORTLET_QUERY_TRANSLATIONS' AND tname.language_code = 'en'
                                       AND tname.lookup_code = 'Effort'
                    LEFT    OUTER JOIN cmn_lookups_v userlov1 ON wip.user_lov1 = userlov1.lookup_code AND userlov1.lookup_type = 'PRTIMEENTRY_USER_LOV1' AND userlov1.language_code = 'en'
                    LEFT    OUTER JOIN cmn_lookups_v userlov2 ON wip.user_lov2 = userlov2.lookup_code AND userlov2.lookup_type = 'PRTIMEENTRY_USER_LOV2' AND userlov2.language_code = 'en'
                    LEFT    OUTER JOIN cmn_lookups_v transtype ON wip.transtype = transtype.lookup_code AND transtype.lookup_type = 'RPT_TRANSACTION_TYPE' AND transtype.language_code = 'en'
                    LEFT    OUTER JOIN cmn_lookups_v costtype ON wip.cost_type = costtype.lookup_code AND costtype.lookup_type = 'LOOKUP_FIN_COSTTYPECODE' AND costtype.language_code = 'en'
                    LEFT    OUTER JOIN ppa_wipapinfo info ON info.id = wip.ppa_wipapinfo_id AND info.voucherno = info.voucherno
WHERE               wip.status = 0
                    AND     inv.id = @NVL@(:param_/data/id/@value,inv.id)
                    AND     inv.id in (select object_instance_id from odfsec_inv_v2 where user_id = 1)
